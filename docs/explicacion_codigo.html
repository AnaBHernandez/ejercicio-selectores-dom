Explicación Línea a Línea del Proyecto

Archivo: script.js

const todosLosParrafos = document.querySelectorAll('#lipsum p');

¿Qué hace?: Busca todos los párrafos dentro de la caja 'lipsum'. Crea una lista para que luego podamos trabajar con ellos de golpe.

parrafo.id = `parrafo-${indice + 1}`;

¿Qué hace?: Es un bautizo automático. En lugar de escribir IDs a mano en el HTML, el código le pone un nombre único a cada párrafo (parrafo-1, parrafo-2...).

evento.preventDefault();

¿Qué hace?: Esta línea es la que "frena" al navegador. Evita que la página se refresque al dar click, permitiendo que veamos nuestro mensaje de alerta.

Archivo: style.css

.important { color: red; font-weight: bold; }

¿Qué hace?: Es el estilo que JavaScript activa con classList.add. El código decide CUÁNDO se pone rojo, pero el CSS dice QUÉ tono de rojo es.

Soberanía Técnica: La Carpeta /docs

He aprendido que los servidores de GitHub no leen bien caracteres especiales. Por eso, mediante terminal, renombré Documentación a docs (minúsculas y sin tilde) para asegurar que el enlace sea universal.
