1. Objetivo del Ejercicio

Localizar elementos en el DOM y manipular sus atributos e identificadores dinámicamente con JavaScript.

2. Manipulación de IDs y Texto (Ej. 1 y 2)

const todosLosParrafos = document.querySelectorAll('#lipsum p');
// Buscamos los párrafos y les asignamos un ID dinámico
todosLosParrafos.forEach((parrafo, i) => {
    parrafo.id = `parrafo-${i + 1}`;
});

// Cambio de texto en botones mediante ID
document.getElementById('btn1').innerText = "Aceptar";

3. Gestión de Clases (Ejercicio 3)

En este punto, he usado classList para aplicar estilos definidos en el CSS externo, siguiendo las buenas prácticas de separación de contenido y diseño. 
// Añadimos la clase 'importante' al primer párrafo
todosLosParrafos[0].classList.add('important');
