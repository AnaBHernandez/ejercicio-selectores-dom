📓 Diario de Aprendizaje: Manipulación del DOM

Desarrolladora: Ana Hernández | Sistema: Ubuntu 24.04

1. Selección de Nodos (Ejercicio 1)

He aprendido que antes de cambiar algo, hay que encontrarlo. He usado dos métodos principales:

  • document.querySelectorAll('#lipsum p'): Sirve para atrapar todos los párrafos de una vez y guardarlos en una lista.
  • document.querySelector('ul li:last-child'): Es muy potente porque permite usar selectores de CSS para encontrar un solo elemento específico (el último de la lista).

2. Modificación de Atributos e Identificadores (Ejercicio 2)

Aquí es donde el HTML cobra vida. He realizado tres acciones clave:

// 1. Asignar IDs automáticamente con un bucle forEach
todosLosParrafos.forEach((p, i) => {
    p.id = `parrafo-${i + 1}`; 
});

¿Qué significa esto? En lugar de ir uno a uno a mano, el código recorre la lista y le pone nombre a cada párrafo automáticamente. Es mucho más eficiente.

También he cambiado el texto de los botones usando .innerText. Es como editar el HTML pero desde el código JavaScript.

3. Gestión de Clases CSS (Ejercicio 3)

Este ejercicio es vital para separar el diseño de la lógica. En lugar de cambiar colores directamente en JS, he usado classList.add('important').

La ventaja: El estilo (colores, fuentes) se queda en el archivo style.css, y JavaScript solo decide cuándo se aplica esa clase.

4. Interactividad y Formulario (El "Click")

Para que la web reaccione al usuario, he usado un EventListener. He aprendido que:

  • evento.preventDefault(): Es obligatorio en los formularios para que la página no se refresque y podamos ver el mensaje de alerta.
  • .value: Es la forma en la que JavaScript "lee" lo que el usuario ha escrito dentro de un cuadro de texto.

🚀 Aprendizaje sobre Despliegue (Soberanía Técnica)

Al principio, mi web daba error 404 en GitHub Pages. Aprendí (por las malas) que los servidores web no entienden bien las tildes en las carpetas. Usando la terminal de Ubuntu, renombré Documentación a docs y moví el style.css a la raíz para que las rutas fueran sencillas y universales.
