📓 Diario de Desarrollo: Reto de Selectores

Fecha: 13 de Abril, 2026 | Entorno: Ubuntu 24.04 LTS

1. Objetivo del Ejercicio

Aprender a manipular el DOM (Document Object Model) usando JavaScript Vanilla para localizar elementos y cambiar sus atributos dinámicamente.

2. El Código JavaScript Explicado

const todosLosParrafos = document.querySelectorAll('#lipsum p');
// Significado: Buscamos todos los párrafos dentro del div con ID "lipsum".

todosLosParrafos.forEach((parrafo, indice) => {
    parrafo.id = `parrafo-${indice + 1}`;
});
// Significado: Recorremos la lista y le asignamos un ID único a cada uno.

3. ¿Qué le explico a Susana?

  • Soberanía Técnica: He separado la lógica en archivos distintos y usado Git para el control de versiones.
  • Uso de forEach: Es más eficiente que un bucle for tradicional para manejar listas de nodos.
  • Manipulación Directa: He cambiado el texto de los botones (Aceptar/Cancelar) usando .innerText.
