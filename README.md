# Ejercicio: Selectores y Elementos del DOM

Este proyecto contiene la resolución de los retos prácticos sobre el DOM.

## 🛠️ Tecnologías
* **JS Vanilla** (ES6+)
* **HTML5/CSS3** Semántico
* **Entorno:** Ubuntu 24.04 LTS

## 📋 Objetivos
1. **Ejercicio 1:** Selección precisa de elementos (párrafos, listas y etiquetas de formulario) usando distintos métodos (`querySelector`, `getElementById`).
2. **Ejercicio 2:** Manipulación dinámica de atributos (IDs únicos mediante bucles) y estilos en línea.
3. **Ejercicio 3:** Gestión de clases CSS y reestructuración de nodos.
